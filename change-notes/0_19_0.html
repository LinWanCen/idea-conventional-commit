0.19.0 (02/08/2021)

Thirty-first alpha release.
Features and refactorings

  • Allowed creation of commit messages through a specialized dialog.
    The dialog can be opened via the Conventional Commit icon near the commit message area (the default shortcut is CTRL+B) and it is pretty straightforward to use. It can also be navigated with assistive technologies.
  • Allowed navigation of inspections results with keyboard shortcuts inside the commit message editor.
  • Improved presentation of commit tokens' documentation.
  • Shown the release changelog version in the tab title inside the What's New dialog.
  • Cleaned-up the source code.
  • Moved minimum supported IDEA release to 202.6397
