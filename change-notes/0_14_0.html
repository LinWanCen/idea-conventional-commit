0.14.0 (13/04/2020)

twenty-first alpha release.
Features and refactoring

  • Implemented a new Provider's filter type, which allow to "keep only selected" Provider
  • Allowed changing completion type (Standard / Template) and Provider's filter type (Keep selected / Hide selected) from the completion pop-up's menu. No need to open Settings for that from now on! ;)
  • For template completion, refactored the "No scope" element to behave better
