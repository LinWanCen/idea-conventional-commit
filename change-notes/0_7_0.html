0.7.0 (27/03/2020)

Tenth alpha release.
New features, refactoring and build changes

  • Updated to Kotlin 1.4-M1
  • Used the Kotlin Standard Library provided by the IDE to reduce the plugin's size
  • Enhanced parsing and completion accuracy of commit tokens
  • Enhanced fluidity of the typing experience (e.g. with parenthesis or colon)
  • Created many platform/code tests for bug prevention
  • Did a substantial code clean-up
