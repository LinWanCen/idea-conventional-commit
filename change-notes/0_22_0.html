0.22.0 (22/03/2023)

Thirty-fifth alpha release.

This release focuses mainly on fixing long-standing bugs, while also improving the user experience in certain areas of the plugin.

The following issues have been fixed:

  • #80 - Don't lowercase recently used tokens in autocompletion
  • #88 - New syntax highlighting doesn't work with git "staging area" enabled
  • #89 - IllegalArgumentException: Requested element count -1 is less than zero
  • #93 - CommitSubjectPsiElement cannot be presented
  • #94 - IllegalArgumentException: Requested element count -1 is less than zero
  • #95 - Structural navigation in Rider 2022.2 is broken with plugin enabled
  • #96 - The plugin breaks the entire usage of Intellij Idea - Remote Development
  • #98 - IllegalArgumentException: Requested element count -1 is less than zero
  • #99 - Can't have Conventional Commit and Return Highlighter installed at the same time
  • #100 - Plugin breaks commit dialog completion from GitToolBox
  • #101 - Clear history button is confusing
  • #103 - Error while pressing CTRL + Z
  • #105 - Seems to break the file autocomplete in commit message for Rider
