The aim of this plugin is to provide completion for conventional commits, also named semantic commits, inside the VCS Commit dialog.


The plugin provides:

  • Standard completion - based on context
  • Template completion - initiated intentionally

It also offers a certain level of customization, which means allowing personalized and sharable default tokens, and attaching additional Providers for the various parts of the commit message: type, scope, subject, body, footer type and footer value.
