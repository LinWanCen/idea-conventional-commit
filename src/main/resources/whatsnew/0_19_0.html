Commit creation dialog

Thanks to patrek's suggestion, we now have a dialog to create commit messages.
The dialog can be opened via the Conventional Commit icon near the commit message area (the default shortcut is CTRL+B) and it is pretty straightforward to use. It can also be navigated with assistive technologies.

Regarding accessibility, I've tried using the IDE with NVDA through keyboard only, and I've noticed we are still lacking a bit both on the Platform itself and on my side.
During development I've spent a lot of time trying to understand how Swing works with a11y and the knowledge I've absorbed for the dialog itself will be applied to the other areas of the plugin.

Documentation presentation

The commit token's documentation presentation has changed a bit. I wanted to display more, and more accurate, information for the proposed token.
I don't know if you'll find it better or worse, so please let me know in the feedback issue on GitHub.

Supported IDE releases

As announced in 0.18.0, I've moved up the minimum supported IDEA release to 202.6397.
This allowed the removal of some workarounds in place for compatibility reasons, and will make future development easier.
